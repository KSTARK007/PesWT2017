if you what to change the height or width of canvas do it in the attribute and not in the CSS file

  1. lines context.lineTo(from,to);
  2. rectangles/square context.rect(100(x_axis),600(y_axis),400(width),300(height));
  3. circles context.arc(x(center_x_axis),y(center_y_axis),r(radius),sa(start_angle),ea(end_angle),true(is(anticlock)))
  4. Text
  5. images
  6. Gradent
