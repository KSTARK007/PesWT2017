bubbling and Capturing

CAPTURING is from window to leaf node(TRUE)

Bubbling is from bottom to Window (FALSE)

  • event.target = it points to the place where the event is invocked first
  • event.currentTarget = it point to the Div where the function is been run on
  • event.stopPropagation() = it stops bubbling or capturing when this function is invoked in the dgive Div
  • event.eventPhase will return 1=in capturing 2=on the clicked event 3=on the bubbling phase
